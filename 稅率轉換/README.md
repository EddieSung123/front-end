### 貸款金額轉換
#### 網址 : 
#### 目的 :   
製作一個貸款轉換器，可以輸入貸款的金額、年利率、年數，並計算出每月應付總金額、應付總金額、每月應付利息，讓有貸款的人，可以輕鬆的知道，每個月應繳納的金額。
#### 技術 :  
**前端框架 : Bootstrap 4、HTML、CSS**  
**程式語言 : JavaScript**  
#### 流程 : 
1. 使用 Bootstrap 4、HTML、CSS呈現出想要的前端框架
2. 讀取貸款的總金額、年利率、年數
3. 給計算Button委派事件，當按下時，套入公式計算
4. 將計算的結果印出來
